Wat is T-Huis UNIX Public Access Server?

T-Huis UNIX Public Access Server, ook wel U-PAST genoemd, is een initiatief van staff en leden van de T-Huis Discord server, de eerste, grootste en gezelligste transgender Discord server van Nederland. Als je hier interesse in hebt, kan je je aanmelden via T-huis.xyz!

Van oudsher werden er UNIX servers aangeboden omdat de kosten voor het hebben van een eigen systeem te duur was voor velen. Omdat UNIX een meerdere-gebruiker systeem is, konden meerdere mensen tegelijk van het systeem gebruik maken. Tegenwoordig kan praktisch iedereen een computer welke vele malen sneller is dan een server van vijftig jaar geleden, echter bestaan Public Access Servers nog steeds. Het kan nu meer gezien worden als een onderdeel van de hacker-cultuur, voor mensen wie graag spelen met computersystemen. Dit betekent echter niet dat er geen nut achter een Public Access Server zit. Men kan bijvoorbeeld websites aanbieden vanaf Public Access Systemen en nog veel meer.

Beschikbare diensten

Deze server is bedoeld om een inclusieve, vrije omgeving te zijn waar men gezamenlijk kan experimenteren met een Unix-achtig systeem. Om dit te verwezenlijken worden er een aantal diensten beschikbaar gesteld, welke het mogelijk maken voor gebruikers om eenvoudig met elkaar te verbinden. Voorbeelden van geleverde diensten zijn:

  • toegang tot een shell-omgeving door middel van SSH
  • web-hosting, inclusief PHP ondersteuning
  • E-mail
  • en meer!
